<h3>Installing</h3>

```java
$ apt update && apt upgrade
$ termux-setup-storage;mkdir $HOME/storage/shared/XNXX-v2
$ apt install git
$ apt install php
$ git clone https://github.com/ArroKM/XNXX-v2
$ cd XNXX-v2
$ php xnxx.php
```
